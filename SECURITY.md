# Security Policy

## サポート対象バージョン

このリポジトリは個人練習用プロジェクトであり、明示的なリリースバージョンは管理していません。最新の `main` ブランチのみがセキュリティ対応の対象です。

## 脆弱性の報告

公開 Issue で脆弱性を報告しないでください。

[GitHub Private Vulnerability Reporting](https://github.com/yabutayukinari/type89/security/advisories/new) から非公開で報告してください。

含めていただきたい情報:

- 影響を受ける箇所（ファイル / エンドポイント / バージョン）
- 再現手順
- 想定される影響範囲
- （任意）修正案・PoC

## 対応の目安

- 受領確認: 7 日以内
- 一次トリアージ: 14 日以内
- 修正・公表: 重大度に応じて個別対応

個人プロジェクトのため SLA は保証しません。ベストエフォート対応とご理解ください。
